3) Fetch the web auth token (the daemon generates one on first launch and
   stores it inside the persistent volume):

     kubectl -n {{ .Release.Namespace }} exec deploy/{{ include "errandd.fullname" . }} -- \
       cat /app/.claude/errandd/web.token

   Append it as ?token=<value> on the first hit to /ui/. The server then sets
   a session cookie so later requests don't need the query string.
